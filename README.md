## 🛠️ 安装方法
### Android
1. 下载 `.apk` 文件到您的设备。
2. 点击安装包完成安装（请确保允许未知来源应用安装）。

### Windows
1. 下载 `.exe` 文件。
2. 双击安装包，根据提示完成安装。

### macOS
1. 下载 `.dmg` 文件。
2. 打开文件并拖动应用图标至应用程序文件夹。
